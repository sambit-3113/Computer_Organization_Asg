#Verilog and Assembly programs  
This Repository contains two folders **Verilog** and **QtSPIM** which contains the verilog and and assembly programs respectively.  

Inside each assignment folder of Verilog there are question folders which contains ".v" file which is the source code and ".vvp" and ".vcd" files which are the Simulation log and VCD waveforms (which is used by waveform viewers such as GTKWave) respectively. The questions and report are attached in each assignment folder.

Inside each assignment folder of Assembly there are question folders which contains ".s" file which is the source code. The questions and report are attached in each assignment folder.
